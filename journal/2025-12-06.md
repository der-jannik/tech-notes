### Best practices

### Compound data types

**Slices:**

- Contagious sequence of elements
- Contegious means uninterrupted -> next to each other in memory

```rust
let number_slice: &[i32] = &[1,2,3,4,5];
```

**String Slices:**

- Fixed size, immutable, without ownership
- Stored on the Stack
- Fast to access
- Only stores a reference to a String literal

```rust
let my_string: String = String::from("Hello, World!")
let my_string_slice: &str = &my_string
```

**Strings:**

- Dynamic size, can be mutable, owned string type
- Allocated on the heap
- Slow to access as it is not fixed

```rust
// Immutable string
let my_string: String = String::from("Hello, world!");

// Mutable string
let mut my_string2 String = String::from("Hello, world!);
```

**Array:**

- Multiple values of one data type (primary or coumpound)
- Need to specify data type and number of values in square brackets `[<data type>; <size of array>]`
- Values are specified in brackets separated by commas `[<val1>, <val2>, ...]`

```rust
let numbers [i32; 5] = [1,2,3,4,5];
```

**Tuples:**

- Multiple values of different data types (primary or compound)

```rust
let myVar1 = ('b', true, 4.5);
let myVar2: (i32, char, float) = (1, 'a', 2.4);
```

### Functions

- To execute a program, the `main()` function is needed
- Else there will be an error
- Custom functions with `fn <my_function>() {}`
- Call by `<my_function>()`
- Rust should be written in snake case
- Hoisting possible: Hoisting means function can be written anyhwere in code, even after where it is called

```rust
fn main() {
    <my_func>(<arg1>);
}

fn <my_func>(<my_param1>: <my_datatype1>) {

}
```

### Expressions, blocks and statements

- Expression: Anything that returns a value
- Statement: Anything that does not return a value
  - Anything that ends with a semicolon
  - Function definitions
  - Control flow statements

```rust
// Expressions
5
true
add(3,4)
if condition val1 else val2 
```

```rust
// Blocks
let _X: i32 = { // _X evaluates to last line in block
    let price: i32 = 5;
    let qty: i32 = 10;
    price * qty
}
```


### Functions with return values
